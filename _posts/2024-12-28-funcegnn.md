This is my class project for CS224W: Machine Learning on Graphs, done in collaboration with Alec Lessing and Matt Wolff. 

The task was to predict whether or not a protein possesed a particular functional property. We used AlphaFold 2 protein structure predictions and structural feature augmentations as an input protein graph. We treated the graph as heterogenous, whereby (source_atom_type, target_atom_type) were treated as unique edge types. We found that distinguishing this heterogenous structure on the protein graph was important. Furthermore, we introduced a Low-rank decomposition, whereby we used a shared weight in the message passing for all edge types and used two low-rank matrices for edge type-specific weights. We used multi-task learning such that each protein function was treated as a separate task with an embedding that we conditioned on. You can follow the details in the Medium post: https://medium.com/@sergiogcharles/funce-gnn-protein-function-prediction-using-multi-task-and-relational-graph-learning-74f972a776f9.
